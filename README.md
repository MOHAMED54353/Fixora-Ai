# Fixora AI 

# Overview
Fixora is an AI-powered platform designed for car maintenance and technical support services.  
It helps users manage car issues, book services, and get AI-based recommendations.

---

## Features
- AI-based car problem diagnosis
- Service booking system
- Technician management
- User dashboard
- Car management system
- Notifications system
- Ratings & reviews

---

## Tech Stack
- React (Vite)
- JavaScript
- CSS
- .NET

---

## Project Purpose
Graduation Project aimed at improving car maintenance experience using AI and smart automation.
